# SmartDoc & Code Review

> Веб-платформа для автоматической проверки Python-кода на соответствие **PEP 8** и Word-документов на соответствие **ГОСТ** (нормоконтроль).

Студент или преподаватель загружает файл — система за секунды показывает ошибки оформления и предлагает исправления, вместо того чтобы вычитывать всё вручную.

## Зачем это нужно

Проверка студенческих работ на оформление — рутина, которая отнимает у преподавателя время, а студент получает обратную связь с задержкой и повторяет одни и те же ошибки. SmartDoc автоматизирует эту проверку:

- **Код** — анализ на соответствие стандарту оформления PEP 8.
- **Документы** — проверка Word-файлов на соответствие ГОСТ (шрифт, отступы, поля, заголовки).

Инструмент универсальный: подходит для любой дисциплины, где сдают код или документы по стандарту.

## Возможности

- ✅ Проверка Python-кода на PEP 8 через `flake8`
- ✅ Проверка `.docx` на соответствие ГОСТ через `python-docx`
- ✅ Автоисправление части ошибок оформления (фиксеры)
- ✅ Веб-интерфейс: загрузка файла и отчёт с ошибками
- ✅ Покрытие тестами

## Запуск локально

```bash
# 1. Клонировать репозиторий
git clone https://github.com/damelonnila/smartdoc.git
cd smartdoc

# 2. Создать и активировать виртуальное окружение
python -m venv venv
source venv/bin/activate      # Windows: venv\Scripts\activate

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Запустить
python run.py                 
```

После запуска открой в браузере `http://localhost:5000`.
